# Relational and Logical Operators

	- Zero - __False__  


Relational Operator | Name | Logical Operator | Name
---- | --- | --- | ---
| == |	Equal to | && | and
| > |	Greater than | \|\| | or
| < |	Less than | ! | not
| != |	Not equal to
| >= |	Greater than or equal to
| <= |	Less than or equal to

# Switch Statement

```C
switch (month) {
    case 2:
        length_of_month = 28 +
        (year%4==0 && (year%100!=0 || year%400==0));
        break;
    case  4:
    case  6:
    case  9:
    case 11:
        length_of_month = 30;
        break;
    default:
        length_of_month = 31;
        break;
}
printf("month=%2d, length_of_month=%2d\n", month,length_of_month);
```

# "for" loop
- Grammar
  ```C
  for (initialize; guard; update){
      statement
  }
  ```
- Example
  ```C
  for (int i = 0; i < 10; i++){
      printf("i = %d\n", i);
  }
  ```

# while() vs. do...while()
- do...while()
  ```C
  int i = 0;
  do{
      printf("i = %d\n", i);
      i++;

  }while(i < 10);
  ```
- while()  
  ```C
  int j = 0;
  printf("j = %d\n", j);
  j++;
  while (j < 10) {
      printf("j = %d\n", j);
      j++;
  }
  ```
