# Array of Strings
Example: 
```C
char *words[10] = {"Algorithms", "are", "fun"};
int i;
for (i = 0; words[i] != NULL; i++) {
    printf("%s\n", words[i]);
}
```
Output:
```C
Algorithms
are
fun
```
